# proctice_frontend_01)コーヒーゼリー紹介サイト
HTML/CSSを用いて制作した、コーヒーゼリーを紹介するWebサイトです。

## 実装で意識した点

### カードデザイン
- 情報を整理し、視認性を高めるためにカード型レイアウトを採用
- ホバー時にカードが浮き上がるアニメーションを実装
- シグニファイアを意識し、「クリックできそう」と感じられるUIを設計
- 将来的には各カードから詳細ページへ遷移できる構成を想定

### レスポンシブ対応
- CSS Gridを採用
- `auto-fit` と `minmax()` を使用し、画面サイズに応じてカード数が自動で変化するレイアウトを実装
- PC・タブレット・スマートフォンで閲覧しやすいデザインを意識

### デザイン
- コーヒーゼリーや純喫茶をイメージしたブラウンを基調とした配色
- サイト全体の世界観に統一感を持たせることを意識
- 余白や色使いによって、落ち着いた雰囲気を表現
